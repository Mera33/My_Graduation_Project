ScATAC-Seq FAQs


Description

ChromoPred which is a CNN-based approach for modeling scATAC-seq data that leverages the DNA sequence information underlying accessibility peaks to generate single-cell chromatin accessibility landscapes across different cell types, and distinguish every peak by its cell types

These are the cell types the user can predict in human :
1- LMPP stands for "lymphoid-primed multipotent progenitors"

2- MPP stands for "multipotent progenitor cell"

3- CLP stands for "common lymphoid progenitor"

4- MEP stands for "megakaryocyte-erythroid progenitor"

5- GMP-A stands for “Granulocyte-Monocyte Progenitor – A”

6- pDC stands for "plasmacytoid dendritic cell"

7- HSC stands for "hematopoietic stem cell"

8- CMP stands for "common myeloid progenitor"

9- UNK stands for “natural killer cell”
